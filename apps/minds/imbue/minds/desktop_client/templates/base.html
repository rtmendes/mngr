{# Base layout: every page extends this. Pulls in the Tailwind Play CDN JS (downloaded once into /_static/tailwind.js and served locally, so this is offline-safe after the first fetch) plus the hand-written tokens.css. The Play CDN script generates utility classes at runtime, so there is no "rebuild when you add a class" step. Blocks: title -- contents. Defaults to "Minds". head -- extra <head> content (page-specific <link>/<meta>). body_class -- classes on <body>. Default is the light page surface. body_style -- inline style on <body>. Used to set --workspace-accent on in-workspace pages so the top stripe picks up the deterministic per-project color. content -- the page body. scripts -- <script> tags emitted just before </body>. Prefer <script src="/_static/foo.js"> over inline JS. #} <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>{% block title %}Minds{% endblock %} {% block head %}{% endblock %} {% block content %}{% endblock %} {% block scripts %}{% endblock %}
